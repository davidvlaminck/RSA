The Cypher query is:
MATCH (k:Wegkantkast {isActief:TRUE})-[:Bevestiging]-(vr:Verkeersregelaar {isActief:TRUE})
WHERE vr IS NOT NULL AND (k.toestand IS NULL OR k.toestand = 'in-ontwerp')
RETURN k.uuid, k.naam
